# Pig Dice

Re-leitura do jogo Pig Dice em C++ para a disciplina de LP I.

# Como Compilar
Para compilar o jogo, abra a pasta src, a seguir, compile utilizando o comando `g++ -std=c++11 main.cpp ex` e então execute o programa com `/ex`.
